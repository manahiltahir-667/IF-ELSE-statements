# Student Grade and Fee Discount Calculator (C++)

## Overview

This is a simple C++ program that calculates a student's **grade** and **fee discount** based on the marks entered by the user. The program uses **conditional statements (if–else)** to evaluate the marks and assign the appropriate grade and scholarship discount.

This project was created as part of my early learning in **C++ programming**, focusing on understanding program logic and decision-making structures.

## Features

* Takes student marks as input
* Assigns a grade based on performance
* Calculates a fee discount according to the grade
* Demonstrates the use of **if–else conditional statements**

## Grade Criteria

| Marks Range | Grade | Fee Discount |
| ----------- | ----- | ------------ |
| 85 – 100    | A     | 50000        |
| 75 – 84     | B+    | 30000        |
| 70 – 74     | B     | 20000        |
| 65 – 69     | C     | 10000        |
| 60 – 64     | D+    | 0            |
| 50 – 59     | D     | 0            |
| Below 50    | F     | No discount  |

## Technologies Used

* C++
* Standard Input/Output Library (`iostream`)

## How the Program Works

1. The user enters their marks.
2. The program checks the marks using a series of **if–else conditions**.
3. Based on the marks, a grade is assigned.
4. A fee discount is displayed according to the grade.
